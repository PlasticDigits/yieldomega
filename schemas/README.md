# Schemas

JSON schemas for onchain metadata structures. Trait keys and types must
stay aligned with their Solidity counterparts (see each schema's `$id`
and the contract struct it references).

**Retired:** collectible NFT metadata schema moved to [`archive/`](archive/) with Arena v2 ([#241](https://gitlab.com/PlasticDigits/yieldomega/-/issues/241)).

See [`CHANGELOG.md`](CHANGELOG.md) for historical semver notes.
